Mystical Rings

An online web catalogue of unique and bizarre rings

The ‘Mystical Rings’ web application is a collection of beautiful, bizarre and unique rings. It hosts ring collections created by incredibly talented and world-renowned designers such as Pamela Love, Mania Mania, Blood Milk Jewels etc. It is a portal for ring lovers who want to discover one-of-a kind pieces that reflect their controversial and unique personalities and thus it gives them the opportunity to browse, view and discover several ring designs, carefully selected and organised.

Users with admin rights can also upload their own personal photos, wearing unique jewellery pieces that they want to share with other unique and alternative people with similar tastes. Normal visitors cannot upload any images, however they are still able to view the users photo collection.

ArtsySocial

A social network designed for artistic people

“Artsy-Social” is a micro social network developed for talented people that simply want to showcase their creative work pieces as well as to form social connections with people from similar backgrounds. The concept of Artsy-Social was inspired by existing popular social media platforms such as Twitter, Facebook, LinkeIn and it was an effort to design a user-friendly platform that would support and offer known to users features but would be also benefited by some unique implementations, no previously attempted. Its brand name derives from the words, “artsy” which is a slang word used to describe people involved in artistic endeavours, and the word “social”.

This web application gives users the ability to register new accounts and then use their personal credentials to log in and access its full content. Users have the ability to showcase and share their personal portfolios along with other personal information with the rest of the world. The main feature, ‘ArtsySocial’ currently provides though is a social media environment where only registered users are allowed to post messages in their personal stream feeds as well as querying other users’ profiles. The application also gives users the option to follow or unfollow people and post messages to their personal stream feeds.

"Rapunzel"

Digital Storytelling website inspired by the 1812 version of “Rapunzel” by the Brothers Grimm set in a retro-futurist universe.
"Rapunzel" is an interactive web experience based on the reimagining of the 1812 version of “Rapunzel” by the Brothers Grimm set in a retro-futurist universe. The storyline bears some resemblance to the original version but contains a significant plot twist that influences the narrative dramatically.

It makes use of the vertical parallax scrolling effect to produce a seamless storyline and facilitate user navigation. It is responsive and employs user-centred design principles. To make the user experience as interactive as possible it employs sound effects on hovering over objects as well as a story-themed soundtrack for each page. It also enables the user to participate into the narrative by selecting their own ending of the story by clicking the appropriate button.

I'm Lost

Android based Mobile Application prototype developed for the elderly

"I'm lost" is a prototype of a mobile application for the Android platform. It has been mainly designed to assist older people and make their lives safer. It provides very straightforward navigation, large font size throughout the application and is very easy to use.

From the main screen users can navigate to all the four different features of the application and each new screen provides a "GO BACK" button for easy redirection to the home page.

The app implements the following features:

  • Ability to store user details including name, age, address and emergency contact number for seniors with memory-related problems
  • Ability to make an emergency call directly from the application
  • Ability to send a pre-defined but editable text message to the emergency contact
  • GPS Facility that enables users to locate themselves in a map (Google Maps integration) and sends their GPS coordinates to the emergency contact in case they are lost or in danger

Technologies Used:

  • Android Studio
  • Java
  • XML
